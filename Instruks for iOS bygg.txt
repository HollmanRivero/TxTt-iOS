============================================================
 TxTt på iPhone — instruks
 Gjelder BÅDE iPhone X (10) og iPhone 15/16 — samme app!
============================================================

VIKTIG Å VITE FØRST
  - Du trenger IKKE to forskjellige filer. Én og samme app
    virker på begge telefonene:
      * iPhone X kjører opptil iOS 16  -> OK (appen krever iOS 13+)
      * iPhone 15/16 kjører iOS 17/18  -> OK
  - Med gratis Apple ID kan du IKKE sende en .ipa-fil til
    telefonene. Hver telefon må kobles til Mac-en din med
    kabel, og appen installeres via Xcode (steg 4).
  - Vil du slippe kabel og dele med andre: betalt Apple
    Developer-konto (~99 USD/år) + TestFlight (se nederst).

STEG 1 — Klargjør prosjektet (allerede gjort hos deg!)
  Terminal:
    cd ~/Documents/GitHub/TxTt-iOS
    bash build-ios.sh
  (ios/-mappen finnes allerede, så scriptet bare oppdaterer
   og åpner Xcode.)

STEG 2 — Logg inn med Apple ID i Xcode (kun første gang)
  Xcode-menyen > Settings > Accounts > "+" > Apple ID.

STEG 3 — Signering
  1. Venstre panel: klikk "App" (øverste, blått ikon)
  2. Fane "Signing & Capabilities"
  3. Huk av "Automatically manage signing"
  4. Team: velg ditt navn ("Personal Team")
  5. Hvis Bundle Identifier klager: endre "com.txtt.app"
     til noe unikt, f.eks. "no.byggsalazar.txtt.hollman"

STEG 4 — Installer på telefonene (gjenta per telefon)
  1. Koble telefonen til Mac-en med kabel
  2. På telefonen: "Stol på denne datamaskinen"
  3. Telefonen: Innstillinger > Personvern og sikkerhet >
     Utviklermodus > PÅ (telefonen starter på nytt)
     (iPhone X med iOS 15 eller eldre: dette steget
      finnes ikke — bare hopp over det)
  4. Xcode: velg telefonen i enhetslisten øverst > trykk ▶
  5. Første gang: telefonen > Innstillinger > Generelt >
     VPN og enhetsadministrering > stol på utvikler-profilen
  Gjenta 1-5 med neste telefon.

BEGRENSNINGER MED GRATIS APPLE ID
  - Appen virker i 7 dager per installasjon — koble til og
    trykk ▶ igjen for å fornye
  - Maks 3 apper samtidig per gratis-konto

OPPDATERE APPEN SENERE
    cd ~/Documents/GitHub/TxTt-iOS
    npm run build && npx cap sync ios
  ...og trykk ▶ i Xcode for hver telefon.

------------------------------------------------------------
VALGFRITT: EKTE .IPA / TESTFLIGHT (krever betalt konto)
  1. Xcode: Product > Archive
  2. Window > Organizer > velg arkivet > "Distribute App"
  3. "TestFlight" = enklest: telefonene installerer via
     TestFlight-appen, ingen kabel, varer 90 dager
  4. "Release Testing" (ad hoc) = gir .ipa-fil for manuell
     installasjon på registrerte enheter
------------------------------------------------------------

RYDDETIPS
  Mappene ModuleCache.noindex, SDKStatCaches.noindex,
  SymbolCache.noindex, Xcode.app-*, App-* og
  Unsaved_Xcode_Document_* i prosjektmappen er Xcode-cache
  som havnet feil. De kan trygt slettes i Finder (jeg har
  lagt dem i .gitignore så de ikke havner på GitHub).
  Det samme kan build-android.bat og build-android.sh
  (de hører hjemme i TxTt-Android-mappen).
